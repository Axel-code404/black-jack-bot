# 🃏 Discord Blackjack BOT

Discordのスラッシュコマンド `/blackjack` でプレイできるブラックジャックゲームBOTです。

テーブル画像・トランプ画像を使用し、視覚的にリアルで分かりやすいゲーム体験を実現しています。

---

## 🎮 主な機能

### 🃏 ブラックジャックゲーム
- `/blackjack` コマンドでゲーム開始
- ヒット / スタンド操作対応
- ディーラー自動処理
- 勝敗判定

---

### 📺 ゲーム専用チャンネル設定機能



# /blackjack channel


このコマンドで **ゲーム実行可能チャンネルを設定**できます。

設定していないチャンネルでは `/blackjack` を実行できないため、

- 他チャンネルへのゲームメッセージ混入を防止
- サーバー運営の整理
- ゲーム専用チャンネルの明確化

が可能です。

---

### 📊 勝敗履歴確認機能

プレイヤーごとの

- 勝利数
- 敗北数
- 引き分け数

を確認できる履歴コマンドを実装しています。
